	INFORMATIONS:

	Cette application a été conçue dans le cadre du projet de Programmation C des étudiants
	de la 2ème année de Licence Informatique de l'Université de Poitiers. Elle a été entièrement
	développée par nous.


========>
	SOURCE DE DEVELOPPEMENT:
	
	Ce jeu a été développé sous Windows 7 avec comme IDE Code Blocks 16.01 .
	Il a été conçu pour tourner uniquement que sous Windows.
	Car nous avons utilisé des commandes système qui sont propres à Windows.
	Aucune librairie externe n'a été utilisée exceptée celles de base (stdio.h,
	stdlib.h et time.h) et celles que nous avons créées.


=======>
	PRE-REQUIS:
	
	Nous vous recommandons d'utiliser Windows: 7, 8 ou 10 pour jouer à ce jeu.
	Car l'une des commandes système utilisée n'est disponible qu'à partir de
	Windows 7 (la commande "TIMEOUT").
	Et surtout veuillez avoir un terminal qui fonctionne correctement pour éviter
	un mauvais affichage (des messages d'erreur de votre terminal).
				
											Amusez-vous!
